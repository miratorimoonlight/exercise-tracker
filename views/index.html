Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

API Endpoints:

  • Log all users: GET /api/exercise/users
  • Log all exercises and filter: GET /api/exercise/log?{userId}[&from][&to][&limit] ({ } = required, [ ] = optional)

from, to = dates (yyyy-mm-dd); limit = number
